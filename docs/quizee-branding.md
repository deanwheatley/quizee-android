# Quizee Branding Guidelines

## Brand Identity
Quizee is a modern, engaging quiz and educational assessment platform that makes learning fun and interactive.

## Color Palette

### Primary Colors
- **Quizee Indigo**: `#6366F1` - Main brand color, used for primary actions and branding
- **Dark Indigo**: `#4338CA` - Used for hover states and dark theme elements

### Secondary Colors
- **Purple**: `#8B5CF6` - Accent color for special features and highlights
- **Green**: `#10B981` - Success states, correct answers, achievements
- **Red**: `#EF4444` - Error states, incorrect answers, warnings

### Neutral Colors
- **Background**: `#F8FAFC` - Light gray background for main content areas
- **Surface**: `#FFFFFF` - White for cards, modals, and elevated surfaces
- **Integration Red**: `#DC2626` - Used specifically for integration/development builds

## Typography
- Clean, readable fonts that work well for educational content
- Good contrast ratios for accessibility
- Support for multiple languages and character sets

## Logo & Icons
- Simple, memorable icon that works at various sizes
- Quiz/education themed imagery (question marks, lightbulbs, etc.)
- Consistent with modern app design trends

## Voice & Tone
- Friendly and encouraging
- Educational but not intimidating
- Celebrates learning and progress
- Inclusive and accessible

## App Themes

### Light Theme (Default)
- Background: Light gray (#F8FAFC)
- Primary: Indigo (#6366F1)
- Text: Dark gray/black for readability

### Dark Theme
- Background: Dark gray/black
- Primary: Lighter indigo (#4338CA)
- Text: Light gray/white for readability

## Usage Guidelines
- Use primary indigo for main navigation and key actions
- Green for positive feedback (correct answers, achievements)
- Red for negative feedback (incorrect answers, errors)
- Maintain consistent spacing and typography
- Ensure accessibility standards are met (WCAG 2.1 AA)

## File Naming Conventions
- Use `quizee_` prefix for brand-specific resources
- Follow Android naming conventions (lowercase, underscores)
- Include size/density suffixes for images (hdpi, xhdpi, etc.)
